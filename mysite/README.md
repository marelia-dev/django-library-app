# Django Library App

Šis repozitorijus skirtas mokomajam Django projektui, sukurtam mokymų "Python → SQL → Django" metu.

### Apie projektą

Tai demonstracinis bibliotekos valdymo projektas, sukurtas Django framework'e.  
Dauguma funkcionalumo buvo kuriama kartu su dėstytoju, o aš kartojau, supratau ir atlikau praktines užduotis.

### Projekto eiga

Projektas buvo kuriamas etapais per visą kursą:

- Modelių kūrimas (Author, Genre, Book, BookInstance, Profile, BookReview)
- Administracinės panelės konfigūracija
- Vartotojų registracija ir autentifikacija
- CRUD operacijos naudojant Class-Based Views (ListView, DetailView, CreateView, UpdateView, DeleteView)
- Darbas su formomis ir TinyMCE
- Nuotraukų apdorojimas (automatinis apkarpymas ir sumažinimas)
- Komentarų sistema prie knygų
- Paieška, filtravimas ir puslapiavimas
- Vertimai į lietuvių kalbą
- Bootstrap 5 dizainas

### Naudotos technologijos

- Python
- Django 6.0
- Bootstrap 5
- SQLite
- TinyMCE
- Crispy Forms
- Pillow (nuotraukų apdorojimui)

### Tikslas

- Praktinis Django framework'o išmokimas
- Darbas su modeliais, views, šablonais, autentifikacija ir autorizacija
- Supratimas, kaip kurti pilnavertį web aplikaciją

### Pastaba

Šis projektas yra mokomasis. Dauguma kodo buvo sukurta kartu su dėstytoju mokymų metu.
